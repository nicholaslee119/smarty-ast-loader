# smarty-ast-loader
a webpack loader to import smarty template as AST
## usage

``` javascript
var ast = require("smarty-ast-loader!./file.tpl");

// or, if you've bound .tpl to smarty-ast-loader
var ast = require("./file.tpl");

```
## test

``` javascript

npm test

```
